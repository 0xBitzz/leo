pub struct Canonicalizer {
    circuit_name: Option<Identifier>,
    in_circuit: bool,
}
Expand description

Replace Self when it is in a enclosing circuit type. Error when Self is outside an enclosing circuit type. Tuple array types and expressions expand to nested arrays. Tuple array types and expressions error if a size of 0 is given. Compound operators become simple assignments. Functions missing output type return a empty tuple.
