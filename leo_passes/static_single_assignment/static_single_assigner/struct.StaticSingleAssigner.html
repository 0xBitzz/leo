Fields

rename_table: RenameTable

The RenameTable for the current basic block in the AST

_handler: &'a Handler

An error handler used for any errors found during unrolling.

counter: usize

A strictly increasing counter, used to ensure that new variable names are unique.

is_lhs: bool

A flag to determine whether or not the traversal is on the left-hand side of a definition or an assignment.

phi_functions: Vec<Statement>

Phi functions produced by static single assignment.

condition_stack: Vec<Expression>

A stack of condition Expressions visited up to the current point in the AST.

early_returns: Vec<(Option<Expression>, Expression)>

A list containing tuples of guards and expressions associated with early ReturnStatements.

Introduces a new AssignStatement for non-trivial expressions in the condition of ConditionalStatements. For example,

  • if x > 0 { x = x + 1 } becomes let $cond$0 = x > 0; if $cond$0 { x = x + 1; }
