The Static Single Assignment pass traverses the AST and converts it into SSA form. See https://en.wikipedia.org/wiki/Static_single-assignment_form for more information. The pass also flattens ConditionalStatements into a sequence of AssignStatements. The pass also rewrites ReturnStatements into AssignStatements and consolidates the returned values as a single ReturnStatement at the end of the function.

Consider the following Leo code.

function main(flag: u8, value: u8) -> u8 {
    if (flag == 0u8) {
        value += 1u8;
        return value;
    } else {
        value += 2u8;
    }
    return value;
}

The SSA pass produces the following code.

function main(flag: u8, value: u8) -> u8 {
    let $cond$0 = flag == 0u8;
    let value$1 = value + 1u8;
    let $return$2 = value$1;
    let value$3 = value + 2u8;
    let value$4 = $cond$0 ? value$1 : value$4;
    let $return$5 = value$4;
    return $cond$0 ? $return$2 : $return$5;
